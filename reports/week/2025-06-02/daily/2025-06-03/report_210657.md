## Common Issues
### 1. Pieces OS connectivity issues across VS Code and Chrome extensions
Users report various connectivity problems with Pieces OS version 11.4.2 on Windows.  VS Code extensions fail to connect, showing empty settings or errors related to custom URLs and version strings. Chrome extensions also experience connection failures and outdated plugin messages.  Potential solutions include verifying network settings, checking Pieces OS and extension versions, and ensuring proper communication between Pieces OS, VS Code, and Chrome extensions.

### 2. Pieces Desktop Application Bugs and Connectivity Issues
Users are experiencing various issues with the Pieces desktop application, including keyboard shortcuts not working on macOS, Chrome plugin connectivity problems on Windows, and LTM repeatedly turning off on macOS.  Troubleshooting steps may include checking for software updates, restarting the application, and ensuring proper plugin versions.

### 3. Pieces OS: Data Persistence, Performance Issues, and LTM Stability
Users report issues with Pieces OS across Windows and macOS, including incomplete data clearing after using "Clear Long Term Memory Data", performance degradation in VSCode while Pieces OS is active, and LTM repeatedly turning off.  Workarounds include manual deletion of the 'com.pieces.os' directory for data clearing and closing Pieces OS to improve VSCode performance.  A more comprehensive data clearing feature and improved LTM stability are desired.

**Related Issues:**
- [#743](https://github.com/pieces-app/support/issues/743): Clear Long Term Memory Data" Does Not Remove Captured Context or Workstream Activity
- [#732](https://github.com/pieces-app/support/issues/732): Performance Issues in VSCode with Pieces OS Active
- [#734](https://github.com/pieces-app/support/issues/734): bug: LTM continues to turn off, requiring me to turn LTM back on periodically

### 4. Pieces OS 11.4.2 bugs on Windows
Users report issues with Pieces OS 11.4.2 on Windows, including memory creation failure via MCP, intermittent connection drops, VSCode performance degradation, and image pasting failures.  A fix for MCP memory creation is expected in 11.4.3. Further investigation is needed for other issues.  Users should update to 11.4.3 when available and report if issues persist.
