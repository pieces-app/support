## Common Issues
### 1. Pieces OS LTM and Workstream Functionality Issues
Users are experiencing problems with the Pieces OS Long Term Memory (LTM) and Workstream features, primarily on macOS.  The LTM fails to return results after extended use or stops recording new activities altogether.  Workstream updates may also be affected.  Clearing LTM data is unsuccessful.  Potential solutions include checking for software updates or reinstalling the application.

### 2. Pieces OS and plugin issues with file dragging, launching, and LLM upgrades
Users report issues with Pieces OS and its plugins on Windows. These include inability to drag files into conversation context, file choosing window appearing in the background, filtering issues, automatic redirection to Pieces support on launch, and difficulty upgrading to the latest LLM version despite Pro subscription. Some users mention using a VPN.  Further investigation is needed to identify the root causes and solutions.

### 3. Pieces OS 12.1.1 Bugs and Feature Requests

### 4. Pieces OS and Desktop App Connection Issues
Users report connection leaks and failures with Pieces Desktop and browser extensions.  The desktop app creates excessive connections to Pieces OS, impacting performance. Browser extensions, even with VPN configurations, struggle to connect.  Temporary solutions include closing the desktop app or restarting Pieces OS.  Issues appear across Windows and involve Pieces OS versions 12.1.1 and 12.2.0, as well as the Pieces for Developers 4.3.1 application.

**Related Issues:**
- [#816](https://github.com/pieces-app/support/issues/816): Pieces Desktop Connection Leak

### 5. Pieces Pro Upgrade and Connectivity Issues
Users are experiencing difficulty upgrading to Pieces Pro, with the checkout process failing to complete.  Additionally, some users report connectivity problems between Pieces OS and browser extensions, even with VPNs configured.  Further investigation is needed to determine the root cause and potential solutions.
